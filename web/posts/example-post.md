# Welcome to My Blog

This is an example blog post written in **Markdown**. This blog is built with Hono and TypeScript.

## Features

- Server-side rendering with JSX
- Markdown parsing with frontmatter support
- Clean, responsive design
- Easy to add new posts

## Code Example

Here's a simple TypeScript example:

```typescript
function greet(name: string): string {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```
